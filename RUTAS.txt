GET
--------
Ruta -> users/ -> Devuelve una lista JSON de todos los usuarios

Response -> 
[
    {
        idusuario,
        idperfil,
        idcarrera,
        username,
        pwd,
        mail
    }
]



Ruta -> users/[numero entero id] -> devuelve un JSON de un usuario dado su idusuario

Request -> idusuario
Response -> 
{
    idusuario,
    idperfil,
    idcarrera,
    username,
    pwd,
    mail
}




POST
------
Ruta -> users/ -> se envia un JSON del tipo USUARIO y devuelve una respuesta si se creo exitosamente
Request -> 
{
    idcarrera,
    username,
    pwd,
    mail
}

Response -> 

{
    response: 'OK'
    message: {
        idusuario,
        idperfil,
        idcarrera,
        username,
        pwd,
        mail
    }
}
